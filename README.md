# wxapp-mall
微信小程序 公司内部用点餐平台

## 相关
文章：[微信小程序之购物车功能](https://github.com/lin-xin/blog/issues/14)

## 前言
GitHub上看到一个类似的项目，就拿来改做一个公司内部用点餐平台，当做小程序开发学习练手了。

## 功能
- [x] 首页
- [x] 分类
- [x] 购物车
- [x] 个人中心
- [x] 商品列表
- [x] 商品详情
- [x] 订单
- [x] 地址管理
